/* TECHNOLOGY */
- Real-time GPU-accelerated fractal rendering using WebGL2 shaders
- Custom LaTeX to GLSL compiler
- FXAA anti-aliasing and adaptive resolution scaling
- MongoDB state persistence
- React frontend with TailwindCSS styling

/* PURPOSE */
JuliaScope is a high-performance web application for exploring 
and visualizing mathematical fractals in real-time. Users can 
input complex mathematical expressions in LaTeX format, which 
are dynamically compiled to GLSL shaders for GPU-accelerated 
rendering.
